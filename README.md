### **DaggerQuest** is an ARPG being built in the [Construct 3 Game Engine](https://editor.construct.net/). It uses pre-rendered images made in [Daz Studio](https://www.daz3d.com/), automated using the ES5-based language [DazScript](http://docs.daz3d.com/doku.php/public/software/dazstudio/4/referenceguide/scripting/start). It also includes a [Selenium](https://www.selenium.dev/) [Java](https://www.java.com/en/) project for automating engine tasks with the [Microsoft Edge WebDriver](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/), and running [JUnit](https://junit.org/junit5/) tests against the [NW.js](https://nwjs.io/) client with [Chrome Driver](https://googlechromelabs.github.io/chrome-for-testing/).
